# Money Drop
Le célèbre jeu Money Drop en logiciel avec une immersion auditive et plus de 100 quizzs avec différentes difficultés.

<br>Le code source du logiciel a été perdu et est actuellement dans un disque dur HS. Je compte, plus grand, récupérer ses données dans un centre de récupération de données.</br>
